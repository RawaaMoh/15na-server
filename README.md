# 15na Server

Central server program of `15na`.

### Requirements

- MATLAB 2015+
- Python 3.6
    - `keras` or `keras-gpu`
    - `numpy`
    - `matlab.engine`
- Node.js 10.x with `yarn`

### How to run

```bash
$ git clone https://github.com/widh/15na-server.git
$ cd 15na-server; yarn
$ # And see this
$ yarn start -- help run
```
